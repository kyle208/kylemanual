:::tip 提醒
默认为【全局设置】下的全局提醒

如需更改分群，请自行下拉选择对应群来设置

**全局设置，优先级2**

**点击需要的群单独设置个性提醒词，优先级1**

**优先级：1>2**

**分群开关内容未设置将在全局设置开关启用的情况下发送，即先判断分群状态，在判断全局状态**
:::

:::caution 注意
旁边的预设按钮，点击后需在【全局设置】下才会显示

微信【进/退群者头像】变量，会单独发送一张图，建议提醒里不要换行

群提醒不联动开关说明：就是不受【入群前】事件影响，建议不用【入群前】功能的开启

退群提醒某人说明：会在群成员退群时，联系主人或某个人，只支持一个，微信为【wxid】，QQ为【QQ号】
:::

:::danger 强烈注意
QQ下，邀请的人不提醒，事件会和其他重复，故不加，不要问为啥不提醒，来提问的都是邀请的状态
:::
