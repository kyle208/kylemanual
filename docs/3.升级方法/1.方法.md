# 二、切换PC框架数据通用
如果需要换框架使用，请将插件数据【kyle】名的目录整体复制替换即可。

myQQ框架数据目录在【MyQQ\data\plugin\kyle】

其他QQ框架数据目录在【框架目录\Config\kyle】
